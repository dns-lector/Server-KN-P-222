Контекст та передача даних до шаблону

Контекст (контекст запиту) - оточення через яке передаються дані для шаблону відображення. З точки зору мови програмування - це dict - структура в стилі ключ-значення. Ключі dict стають змінними контексту.

Дані передаються через інструкцію template.render(context, request), у шаблоні дані доступні через спец. синтаксис {{}}: x = {{x}}, str = {{str}}

{% for key, value in all.items %} {{ key }}: {{ value }}
{% endfor %}
